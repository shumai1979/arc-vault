# ArcVault

Yield vault on Arc Testnet powered by Uniswap v4 God Hook + AI rebalancing.

## What it does
- Deposit native USDC into the vault
- Earn yield on every block
- AI dynamically rebalances liquidity
- Uniswap v4 God Hook optimizes fees automatically

## Built for Arc Testnet
- Uses native USDC as gas (no token approval needed)
- Live on Arc Testnet ahead of mainnet launch
- Part of the Arc ecosystem

## Tech Stack
- React + TypeScript + Vite
- Uniswap v4 (God Hook)
- AI rebalancing engine
- Arc Testnet native USDC
